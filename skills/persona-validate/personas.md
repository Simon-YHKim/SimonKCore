# 일반 사용자 페르소나 라이브러리 (persona-validate)

## 사용 규칙
- 도메인·대상에 맞춰 샘플(예: 개발 도구 검증=U04·U08·U09 중심 + 저테크 1~2; 소비자 앱=폭넓게).
- **나이 ≠ 테크 수준**(U10 47세 저테크). age-only 자동감지 금지, '테크 자신없음' 신호 별도 수집.
- 각 페르소나는 "이 사람이 자기 목표로 들어와 끝까지 가는가"를 본다. 막히는 정확한 지점·이유를 남긴다.
- 전문가 패널(experts.md)과 함께 구성 — 사용자=완주/접근성, 전문가=정확성/리스크.
